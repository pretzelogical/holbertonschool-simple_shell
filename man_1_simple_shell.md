# NAME
**hsh**- simple shell that can run programs

# SYNOPSIS
**hsh**

# DESCRIPTION
**hsh** searches PATH variable for program and if it finds it, runs it with supplied arguments and environment variables.

# BUILT-INS

exit

: Exits the shell

env

: print the current environment of the shell
